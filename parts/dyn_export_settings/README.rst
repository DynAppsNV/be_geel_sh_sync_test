=======================
Dynapps Export Settings
=======================

The module allows you to export the configuration of the Odoo instance.
The settings are exported in a zip file, which contains the
configuration files in the format of the Odoo module. The exported
configuration can be put in the code base. This way the configuration
can be versioned and deployed with the code.
