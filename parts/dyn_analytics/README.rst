========================
Odoo Analytics | Dynapps
========================

This module collects general info about this Odoo installation and sends
it to Dynapps.

The data being sent can be viewed at *Settings -> Technical (Debug) ->
Dynapps Analytics*.

To configure this module, you need to:

-  Set the **dynapps_analytic_endpoint** system parameter if needed.
