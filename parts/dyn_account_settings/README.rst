========================
Dynapps Account Settings
========================

This module adds missing features in Odoo account settings.

It allows to easily configure default values for the following fields:

-  Partner related fields:

   -  Default Account Receivable
   -  Default Account Payable

|image1|

These values are stored in the User-defined Defaults
